# 贵州盖浇面/饭 · 菜品研发过程记录表

基于 Streamlit 的菜品研发记录工具，支持宫保/泡椒双味型，接入 DeepSeek AI 辅助分析。

## 功能

- 📋 **双味型支持**：宫保（糊辣荔枝）+ 泡椒（酸辣）
- 📝 **多版本对比**：V1.0 → 终版定型，一张表纵向记录全过程
- 🔍 **品鉴记录**：观感、香气、味型、口感、芡汁、搭配主食
- 🤖 **DeepSeek AI 助手**：配方分析、版本对比、优化建议
- 📥 **导入/导出**：JSON 格式保存完整数据，CSV 导出配方表
- 🖨️ **打印优化**：A4 横向打印
